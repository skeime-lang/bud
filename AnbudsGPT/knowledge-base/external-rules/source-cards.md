# Kildekort for anbudsregler

Kartleggingsdato: 2026-06-11  
Scope: offentlige/kommunale anskaffelser, særlig bygg, anlegg og eiendom  
Status: kildegrunnlag, ikke juridisk vurdering

Alle kort er markert for fagverifisering før bruk i konkrete tilbud. Kort med `2026-07-01` er versjonsfølsomme fordi regelendringer er varslet eller vedtatt rundt denne datoen.
